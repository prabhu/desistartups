== Description ==

<p>This Plugin is an enhancement for Wordpress's comment function. It enables users to reply on a exist comment, and the discussion will be displayed threaded or nested.</p>

<p>这是一个针对Wordpress评论功能的增强插件。本插件让用户能够对已有评论进行回复讨论，并将结果嵌套或成串显示。</p>

<h3>Features</h3>
<ul>
<li>One can reply on any exist comments.</li>
<li>The discussion will be displayed nested or threaded. </li>
<li>Easy to install. No hacking on Wordpress or your theme is needed. </li>
<li>W3C compatible. </li>
<li>Customizable HTML / PHP / CSS on admin section. </li>
<li>AJAX support, enables to comment without reloading the whole page. </li>
<li>Choose AJAX or not freely.</li>
<li>Get notification by email when reply is available. </li>
<li>Can reply or post comments in wp admin.</li>
</ul>

<h3>功能特性</h3>
<ul>
<li>用户可以对已有评论进行回复讨论</li>
<li>嵌套或成串显示相关讨论。</li>
<li>无需对Wordpress和主题进行修改，便于安装。</li>
<li>W3C兼容。</li>
<li>可于管理后台自定义的HTML、PHP和CSS。</li>
<li>支持AJAX，无需刷新整个页面即可留言。</li>
<li>可自由选择是否使用AJAX效果。</li>
<li>评论有回复时电邮通知原评论人</li>
<li>后台可以回复或发表评论.<li>
<ul>

<h3>Requirement to themes</h3>
<p>If you wish to use AJAX, the comment objects of your theme must has an ID like “comment-xxx”. It's a standard of Wordpress themes now, yet some themes may still not meet this requirement.</p>

<h3>对主题的要求</h3>
<p>如果需要使用AJAX，在主题中的评论的对象必须依从Wordpress主题标准，有“comment-xxx”这个ID，否则将无法使用AJAX功能。目前仍有少数主题不支持。</p>

== Installation ==
<ol>
<li>Download and unzip the Plugin package. </li>
<li>Rename "wordpress-thread-comment" folder to "wp-thread-comment"</li>
<li>Upload it to Wordpress Plugin folder, '*WORDPRESSROOT*/wp-content/plugins/'</li>
<li>Activate from Plugin page in the Wordpress site admin. </li>
<li>Have a check if there is a “reply” link on your exist comments. </li>
<li>Other configurations like AJAX, HTML/CSS and more can be found in the option section in the site admin. </li>
</ol>

<h3>安装</h3>
<ol>
<li>下载并解压
<li>将"wordpress-thread-comment"文件夹更名为"wp-thread-comment"。</li>
<li>将此文件夹上传到Wordpress的插件目录：'Wordpress根目录/wp-content/plugins/'。</li>
<li>在后台的插件管理处激活。</li>
<li>检查已有评论下是否出现了“回复此评论”的连接。</li>
<li>其它设置，如AJAX和HTML/CSS等请移步后台的设置区。</li>
</ol>
